AUTENTICAÇÃO E GERENCIAMENTO DE USUÁRIOS

Backend (NestJS):
@nestjs/jwt - Geração e validação de tokens JWT para autenticação
@nestjs/passport - Middleware de autenticação e proteção de rotas
passport-jwt - Estratégia JWT para validação de tokens
bcrypt - Criptografia e hash de senhas
class-validator - Validação dos dados de entrada (DTOs)
@nestjs/mongoose - Conexão e modelagem com MongoDB
mongoose - ODM para manipulação do banco de dados

Frontend (Next.js):
next-auth - Sistema completo de autenticação e sessões
axios - Cliente HTTP para chamadas à API
jwt-decode - Decodificação de tokens JWT no cliente
react-hook-form - Gerenciamento de formulários de login/registro
zod - Validação de schemas e dados do formulário
cookies-next - Gerenciamento de cookies para tokens

INTERNACIONALIZAÇÃO:
next-intl - Sistema de internacionalização (i18n)
@formatjs/intl-localematcher - Detecção e correspondência de idiomas
negotiator - Negociação de preferências de idioma
